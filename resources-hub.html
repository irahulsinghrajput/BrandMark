📖 How to Save Pages as PDF

🪟 Windows & Mac

  1. Open any resource page or guide
  2. Press Ctrl+P (Windows) or Cmd+P (Mac)
  3. Select "Save as PDF" from the destination dropdown
  4. Choose your download location
  5. Click "Save"

📱 Mobile Devices

iPhone/iPad: Tap Share button → Print → Pinch to zoom in → Tap Share → Save to Files

Android: Tap menu (3 dots) → Print → Select "Save as PDF" → Save

Or save the page directly using your browser's download feature

💡 Pro Tip: All pages are print-friendly! You can customize margins, page size, and printing format before saving.
