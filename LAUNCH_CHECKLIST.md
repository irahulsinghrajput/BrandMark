## 📞 Quick Troubleshooting

### Q: Courses not showing?
**A:** Click browser refresh. Verify MongoDB is connected via `/test-connection.js`

### Q: Admin dashboard blank?
**A:** Check browser console for errors. Verify API base URL is correct.

### Q: Payment not working?
**A:** Verify Razorpay keys in `.env`. Check webhook endpoint.

### Q: Can't access admin page?
**A:** Ensure you're using the full URL with `/admin-enrollment-dashboard.html`
