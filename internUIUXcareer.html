UI/UX Intern

Full-time Hybrid Paid Internship

Job Description

Obsessed with user journeys? Help us map out intuitive interfaces. You'll be conducting user research, creating wireframes, and prototyping designs that minimize friction and maximize delight. We are looking for a UI/UX Intern who is passionate about creating seamless digital experiences.

Key Responsibilities

  • Create wireframes, storyboards, user flows, and process flows.
  • Design user interfaces for web and mobile applications.
  • Conduct user research and usability testing.
  • Collaborate with developers to ensure accurate implementation of designs.
  • Stay up-to-date with the latest UI/UX trends and technologies.

Required Skills

  • Proficiency in design tools like Figma, Sketch, or Adobe XD.
  • Understanding of user-centered design principles.
  • Basic knowledge of HTML/CSS is a plus.
  • Strong portfolio demonstrating UI/UX projects.
  • Excellent communication and teamwork skills.

What We Offer

  • Opportunity to work on diverse projects.
  • Mentorship from senior designers.
  • Creative and collaborative work environment.
  • Potential for full-time employment.
